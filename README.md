# Front-End Lab's

A front-end lab's that helps you build fast an modern web and mobile apps.

http://frontlabs.com.br

## Get the kit

```
$ git clone https://github.com/alexandesigner/frontlabs.git && cd frontlabs
```

## Installation

### If Mobile

Go to folder.

```
$ cd project/mobile
```

Install all dependencies.

```
$ npm install
```

Running.

```
$ ionic serve
```

### If Browser

Go to folder

```
$ cd project/web
```

Install all dependencies.

```
$ npm install
```

Running.

```
$ gulp
```

###Requirements
* node
* npm
* gulp
* bower
* ionic
* angular
