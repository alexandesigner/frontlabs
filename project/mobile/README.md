# Front-End Lab's Mobile (ionic project)

A front-end lab's that helps you build fast an modern mobile apps.

## Get

```
$ git clone https://github.com/alexandesigner/frontlabs.git && cd frontlabs
```

## Installations

Go to folder.

```
$ cd project/mobile
```

Install all dependencies.

```
$ npm install
```

Running.

```
$ ionic serve
```
