# Front-End Lab's Web

A front-end lab's generator that helps you build fast an modern web applications.

## Get

```
$ git clone https://github.com/alexandesigner/frontlabs.git && cd frontlabs
```

## Installations

Go to folder.

```
$ cd project/web
```

Install all dependencies.

```
$ npm install
```

Running.

```
$ gulp
```
